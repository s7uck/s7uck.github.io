Introduzione

C++ è un superset di C, un linguaggio di programmazione compilato, procedurale a basso livello. C++ nasce per aggiungere le classi a C, che non le supporta, ma anche una ampia libreria standard, completa di funzioni di aiuto nella manipolazione dei numeri, degli array, del testo e così via. Ecco un esempio di codice C++: // main.cpp #include <iostream> int main() { std::cout << "Hello world!"; return 0; } Analizziamo il codice riga per riga.
  1. La prima linea è un commento. Non è importante per l'esecuzione del codice, quindi andiamo avanti.
  2. La seconda linea dichiara una funzione. Qui specifichiamo il tipo di ritorno della funzione: in questo caso, la nostra funzione ci restituisce un numero, che in C++ è chiamato int.
  3. La terza linea inserisce una stringa in std::cout:
    • :: è l'operatore di scope, ossia ci permette di accedere agli elementi di una libreria. In questo caso recuperiamo cout da std
    • l'operatore << è l'operatore di inserimento: inserisce un valore all'interno di std::cout
    • la linea termina con ; come tutte le istruzioni C++
  4. La quarta linea restituisce un valore, in questo caso numerico, 0. Nella funzione main(), ossia quella eseguita all'inizio del programma, un valore di 0 significa uscita senza errori, mentre 1 o superiore indica una qualche forma di errore.
Il programma viene poi passato a un qualunque compiler per trasformarlo in un eseguibile, che possiamo poi avviare. Il file si chiamerà, in questo caso, main.out.
